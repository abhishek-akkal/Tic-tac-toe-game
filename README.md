# 🎮 Tic Tac Toe

A simple and interactive Tic Tac Toe web app built using HTML, CSS, and JavaScript.

## 🚀 Features

* 2-player gameplay (X vs O)
* Smooth UI interactions
* Restart / Reset option
* Responsive design

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
